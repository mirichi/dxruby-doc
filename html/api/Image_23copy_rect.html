INDEX > Image > copy_rect

Image#copy_rect(x, y, image, x1=0, y1=0, width=image.width, height=image.height)

Description

selfのオブジェクトのx/yの位置に、imageで指定したオブジェクトのx1/y1/width/heightで指定した部分をコピーします。
α値を無視した単純コピーです。
自分自身にコピーすることはできません。

X座標

image Image

X1座標
y1 Integer
Y1座標
width Integer
横幅
height Integer
高さ
