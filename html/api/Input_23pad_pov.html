INDEX > Input > pad_pov

Input#pad_pov(pad_number=0)

Description

アナログパッドのPOV(ハットスイッチ、通常のスティック付パッドの場合は十字ボタン)の入力を角度の100倍の値(0～335999)で返します。
十字ボタンの場合は4500ごとの値になります。

Parameters

pad_number Integer
パッド番号

Returns

Integer
