INDEX > Image > draw_font

Image#draw_font(x, y, text, font, color=[255,255,255])

Description

Win32APIでビットマップを取得し文字を描画します。
どのサイズでもアンチエイリアスがかかりますが、改行文字を入れても改行されません。
後方互換性のためのメソッドであり、Image#draw_font_exの利用が推奨されます。

Parameters

x Integer
X座標
y Integer
Y座標
text String
書き込む文字列
font Font
フォントオブジェクト
color Array
RGB色配列
ARGBでも指定可能ですが、現在のバージョンではAは無視されます。
