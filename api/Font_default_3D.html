INDEX > Font > default=

Font.default=(font)

Description

デフォルトフォントを変更します。
デフォルトフォントとは言っても描画時にフォントを省略することはできません。
いちいちフォントを生成せずにFont.defaultと指定できるというだけです。

Parameters

font Font
Fontオブジェクト
