Sound#play

Description

音を再生します。
start、loopstart、loopend、loopcountをあらかじめ設定しておくと、部分ループなども可能となっています。
同時にいくつでも再生できます。
Sound#loopCount=で設定しない限り、midファイルは無限ループ、wavファイルは1回の再生となります。
