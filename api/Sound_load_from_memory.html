Sound.load_from_memory(string, type)

Description

Stringオブジェクトでファイルデータを渡してSoundオブジェクトを生成します。
typeはTYPE_MIDIとTYPE_WAVの定数で渡します。

Parameters

string String
音のバイナリデータ
type Constant
音の種類(TYPE_MIDI/TYPE_WAV)

Returns

Sound
