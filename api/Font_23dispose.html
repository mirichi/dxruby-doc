Font#dispose

Description

selfのフォントリソースを解放します。
使わなくなったデータをdisposeしておくと、PCのメインメモリを節約することができ、またGC時の負荷も軽くなります。
dispose後、そのオブジェクトのメソッドを呼び出すと例外が発生します。
