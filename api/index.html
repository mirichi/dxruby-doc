DXRuby 1.4.1 Reference Manual

API INDEX

module Input
キーボード・ゲームパッド・マウスの入力を扱います。
module Window
ウィンドウ生成、グラフィック描画、画面更新を扱います。
class Font
フォントを扱うクラスです。
class Image
画像を扱うクラスです。
class RenderTarget
画像イメージを持ち、Windowモジュールと同じ方法で描画ができ、ハードウェア描画の対象に指定できるクラスです。
class Shader
シェーダ言語であるHLSLを使ってプログラムを記述し、画像の描画時にGPU上で実行することで高速なエフェクトを実現するクラスです。
class Shader::Core
HLSLから実行バイナリを生成・保持するクラスです。
class Sound
BGMや効果音を外部ファイルから読み込んで扱うクラスです。
class SoundEffect
プログラム上で効果音を生成するクラスです。
class Sprite
描画位置、描画パラメータ、画像データを保持し、自分自身を描画する機能を持つ描画プリミティブクラスです。
