INDEX > Image > save

Image#save(path, format=nil)

Description

selfの画像を指定のフォーマットで画像ファイルとして保存します。
省略するとファイル名の拡張子から自動判別されます。判別に失敗するとPNGになります。

Parameters

path String
ファイルパス
format Constant
保存形式（FORMAT_JPG/FORMAT_PNG/FORMAT_BMP/FORMAT_DDS）
