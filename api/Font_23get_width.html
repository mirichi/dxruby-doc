Font#get_width(text)

Description

selfのフォントで描画した場合に横幅が何pixelになるかを計算して返します。
プロポーショナルフォントの場合に描画サイズを計算したい時に使います。
このメソッドでは改行文字は無視されます。

Parameters

text String
描画対象の文字列

Returns

Integer

Note: イタリック体にすると右に傾く分だけ幅は広がりますが、その増分は取得できません。
