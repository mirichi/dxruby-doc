INDEX > RenderTarget > draw_shader

RenderTarget#draw_shader(x, y, image, shader, z=0)

Description

シェーダ処理を適用して描画します。

Parameters

x Integer
X座標
y Integer
Y座標
image Image|RenderTarget
描画元の画像オブジェクト
shader Shader
適用するShaderオブジェクト
z Integer|Float
描画順。
小さいほど奥になり、同じ値の場合は最初にdrawしたものが一番奥になります。
