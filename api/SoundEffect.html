class SoundEffect

Description

プログラム上で効果音を生成するクラスです。
周波数とボリュームを指定時間単位で変化させることができます。
具体的な使い方はチュートリアルを参照してください。

Class Methods

new
効果音を生成して返します。

Instance Methods

disposed?
selfがdisposeされている場合に真を返します。
add
生成済みの効果音に、別の効果音を重ねます。
play
効果音を再生します。
stop
音の再生を止めます。
save
音のデータをwav形式で保存します。
