INDEX > Image > draw

Image#draw(x, y, image, x1=0, y1=0, width=image.width, height=image.height)

Description

selfのオブジェクトのx/yの位置に、imageで指定したオブジェクトのx1/y1/width/heightで指定した部分をコピーします。
α値を考慮して半透明演算されます。
自分自身にコピーすることはできません。

Parameters

x Float
パーリンノイズを求めるx座標を指定します。
y Float
パーリンノイズを求めるy座標を指定します。
image Image
Imageオブジェクト
x1 Integer
X1座標
y1 Integer
Y1座標
width Integer
横幅
height Integer
高さ
