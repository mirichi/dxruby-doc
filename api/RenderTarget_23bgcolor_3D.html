INDEX > RenderTarget > bgcolor=

RenderTarget#bgcolor=(color)

Description

画面のクリア（RenderTarget#update時、描画処理前に自動的にされます）時に使う色を変更します。

Parameters

color Array
ARGB色配列もしくはRGB色配列
