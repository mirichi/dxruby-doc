INDEX > Window > draw_line

Window#draw_line(x1, y1, x2, y2, color, z=0)

Description

画面に(x1, y1)～(x2, y2)を結ぶ線を引きます。

Parameters

x1 Integer
開始X座標
y1 Integer
開始Y座標
x2 Integer
終了X座標
y2 Integer
終了Y座標
color Array
RGB色配列
z Integer|Float
3D的な奥行きではなく、描画順です。
小さいほど奥になり、同じ値の場合は最初にdrawしたものが一番奥になります。
