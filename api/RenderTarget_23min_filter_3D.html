RenderTarget#min_filter=(filter)

Description

draw_scaleやdraw_exのscale_x/yキーで縮小する場合に使用されるフィルタを変更します。
この設定はWindow.scale=には反映されません。

Parameters

filter Constant
以下のフィルタ定数
TEXF_POINT 単純拡大・縮小フィルタ
TEXF_LINEAR 線形補間フィルタ（デフォルト）
TEXF_GAUSSIANQUAD 4-サンプルガウスフィルタ
