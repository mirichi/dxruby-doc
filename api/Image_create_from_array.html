Image.create_from_array(width, height, colors)

Description

色情報を格納した配列からImageオブジェクトを生成して返します。
配列はalpha, red, green, blueの４つをピクセル数分並べたものです。
例)32×32ピクセルの白い四角は[255, 255, 255, 255] * 32 * 32と指定します。

Parameters

width Integer
横幅
height Integer
高さ
colors Array
ARGB色配列を格納した配列

Returns

Image
