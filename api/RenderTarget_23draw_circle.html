INDEX > RenderTarget > draw_circle

RenderTarget#draw_circle(x, y, r, color, z=0)

Description

画面の(x, y)を中心に半径rの円を描画します。

Parameters

x Integer
X座標
y Integer
Y座標
r Integer
半径
color Array
RGB色配列
z Integer|Float
描画順。
小さいほど奥になり、同じ値の場合は最初にdrawしたものが一番奥になります。
